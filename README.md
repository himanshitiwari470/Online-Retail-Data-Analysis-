This project performs Exploratory Data Analysis (EDA) on an Online Retail dataset containing customer transaction records. The dataset includes details such as invoice number, product information, quantity, price, customer ID, and country.

The main goal is to analyze sales trends, customer purchasing behavior, and product performance to generate useful business insights. Data cleaning and preprocessing are performed to handle missing values and ensure data quality.

Using Pandas, Matplotlib, and Seaborn, various visualizations are created to understand top-selling products, country-wise sales distribution, and revenue patterns over time.

Overall, this project helps in transforming raw retail data into meaningful insights for better business decision-making.
